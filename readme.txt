== Description ==

Relish the perfection of RelicWP!(https://relicwp.com/) RelicWP is a lightweight highly customizable theme. Whether you want a blog, portfolio, business, or eCommerce, RelicWP is just what you need. To use this fast, responsive, and SEO-friendly theme to its fullest extent you need to have RelicWP helper. Helper lets you add additional features to your RelicWP theme.
Features include—
<ul>
<li>widgets</li>
<li>activate/deactivate the customizer sections</li>
 
<li>meta boxes</li>
<li>import/export</li>
 
<li>enable/disable the theme’s scripts and styles</li>
<li>Options to activate premium features</li>
</ul>
Before using RelicWP helper, make sure that you have installed RelicWP theme first.

== Installation ==

1. Upload `relicwp-helper` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Ready to use!

== Frequently Asked Questions ==

= I installed the plugin but it does not work =

This plugin will only work with the [RelicWP](https://relicwp.com/) theme.
